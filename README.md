# libxml2-deploy

Build, test and deploy scripts for the [XML library](http://xmlsoft.org/) for CODE-RADE.

# Versions

Versions built :

  - 2.9.3
  - 2.9.4

# Dependencies

LibXML is built with bindings for

  * icu
  * lzma
  * python

These modules are added as dependencies
